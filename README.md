# Log Analyzer (Java)

A **Log Analyzer project in Java** that reads a log file and separates entries into `INFO`, `WARNING`, and `ERROR` categories.  
The project demonstrates **File Handling**, **Exception Handling**, and **Buffered I/O** in Java.

---

## ✨ Features
- Reads from a `Report.log` file
- Separates logs into:
  - `info.log`
  - `warning.log`
  - `error.log`
- Generates a summary report (`report.txt`) with counts

---

## 🛠️ Tech Stack
- **Java 24**
- **File Handling (BufferedReader/Writer)**
- IntelliJ IDEA

---

## 📂 Project Structure
- `LogAnalyzer.java` → Reads and categorizes logs  
- `ReportWriter.java` → Creates a summary report  
- `Main.java` → Entry point for execution  
- `Report.log` → Input log file
